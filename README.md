# GlassTailwind
TailwindCSS practice project

## Init project
```
npm install -D tailwindcss
npx tailwindcss init
```

## Build CSS and watch the change
```
npx tailwindcss -i ./src/input.css -o ./src/dist/main.css --watch
```
